# Michelle – Personal Portfolio

- A clean, modern, and fully responsive single-page portfolio website built with **HTML, CSS, and JavaScript**.

- Designed to showcase my work as a passionate web developer based in **Nairobi, Kenya**.


## ✨ Features

- Fully responsive design (mobile + desktop)
- Smooth scrolling navigation
- Modern project showcase (To-do List, Weather App, Calculator)
- Working contact form with real email delivery (powered by EmailJS)
- Clean and minimal UI with Inter font
- Hamburger menu for mobile
- Success message after form submission

## 🛠️ Technologies Used

- **HTML5**
- **CSS3** (Flexbox + Grid)
- **Vanilla JavaScript**
- **EmailJS** (for contact form)
- Google Fonts (Inter)

## 📂 Projects Featured

- **To-do List App** — Task manager with local storage, add/edit/delete functionality
- **Weather App** — Real-time weather with location detection and forecasts
- **Calculator** — Modern calculator with dark mode and operation history

## 📬 Contact Form

The contact form is fully functional and sends messages directly to my email using **EmailJS**.  
No backend required.
